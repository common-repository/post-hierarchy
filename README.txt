== Description ==
A handy Plugin that help you to set a post to child post of another post. which let you create a sub posts, numbered posts etc.
Same as page, You can make any post a child post of another post and setup hierarchy. This plugin is developed by [Codelab7](https://codelab7.com)


== Frequently Asked Questions ==

= Permalink is not working. What should I do? =

Go to Settings>Permalinks and update twice, and check.

= Do child posts will be looped as other posts =

Yes, for now. But in future releases we'll provide the option to manage from admin panel

== Requirements ==

* WordPress 5.0 or greater and PHP 7.0 or up required
